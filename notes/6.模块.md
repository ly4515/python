```python
'''
__init__.py文件
当导入包的时候，默认调用__init__.py文件
作用：
1.当导入包的时候，把一些初始化的变量，函数，类定义在__init__.py文件中
2.此文件中函数，变量等的访问，只需要通过包名.函数。。。
3.结合__all__=[可以通过*访问的模块]
'''

from 模块 import *     # 如果模块没有定义__all__ 模块中所有内容都可以访问，
                      # 如果定义了__all__，只有__all__ =['','']列表中的可以访问
from 包 import *      # 表示该包中内容（模块）是不能访问的，需要在__init__.py文件中定义__all__=[可以通过*访问的模块]

'''
由于架构不当，可能会出现模块之间相互导入，避免产生循环导入的方法
1.重新架构
2.将导入的语句放到函数里面
3.把导入语句放在模块最后
'''
```
